
===============================================================================

Project Readme

===============================================================================

Setup project in Eclipse:

1. Eclipse -> File -> New -> Project... -> Android Project, choose "Create 
    project from existing source".
2. Choose Android 2.2 or SDK API level 8.
3. Right click the project to choose "Properties", choose "Java Build Path" in 
    the left columne, and "Source" tab in the right columne, add the following 
    folders:
        * account_sync/src
        * account_sync/SyncML
        * notification/src
4. In the "Libraries" tab, "Add Jars" to add needed jar lib from /libs folder. 
5. You may want to add your sub-project or jar lib in the similar above manner.
